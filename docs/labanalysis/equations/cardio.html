252        self,
253        power: float | int,
254        bodyweight: float | int,
255        grade_level: int = 8,
256    ):

258        predict the exercise VO2 in ml/kg/min.
259
260        Parameters
261        ----------
262        power: int | float
263            the power output in W
264
265        bodyweight: float | int

267

283            - 0.2821 * incline
284            + 0.0085 * incline**2
285        )

470        predict the exercise VO2 in ml/kg/min.
471
472        Parameters
473        ----------
474        power: int | float

500        self,
501        vo2: float | int,
502        power: float | int,
503        cadence: float | int,
504    ):

506        predict the bodyweight from vo2 and power
507
508        Parameters
509        ----------
510        vo2: float | int
511            the vo2 in ml/kg/min
512
513        power: float | int
514            the power output in W
515
516        cadence: float | int
517            the cadence in spm

521        bodyweight: float

534                - 23697.453729 / cadence**2

541        self,
542        vo2: float | int,
543        bodyweight: float | int,
544        cadence: float | int,
545    ):
546        """
547        predict the power output from vo2 and bodyweight

549        Parameters

551        vo2: float | int
552            the vo2 in ml/kg/min

554        bodyweight: float | int
